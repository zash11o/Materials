### WSL2（Ubuntu-20） に cron 仕込む

- やりたいこと：単に定期的に自宅環境のインターネット速度取りたいだけ。
  - んが、デフォだと Cronの自動起動ができない…
    - 対策１：`.bashrc` に仕込む
      - 一回ログインするの面倒...
    - 対策２：Windowsのタスクスケジューラに仕込む
      - こっちやな( ﾟДﾟ)y─┛~~

- コンパネ→...→タスクスケジューラ起動
  - 以下のような感じで
    - トリガー： Windows のログオン時
    - 操作タブ：
      - プログラム： wsl
      - 引数： -u root -- service cron start
    - とりあえず 1分 ぐらい遅延させる
      - いつからwslがコマンド発行受け付けるかわからん...
